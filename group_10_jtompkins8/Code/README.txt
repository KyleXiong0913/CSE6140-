The main executable is Program.java

Compile by using the command
javac *.java

Run by using the command
java Program -inst <filename> -alg [BnB|Approx|LS1|LS2] -time <cutoff in seconds> -seed <random seed>

The script run.sh has been provided to run all the inputs through all the algorithms
